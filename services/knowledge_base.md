# How to Create a Knowledge Base Article

Knowledge Base (KB) is an OIT-owned application that hosts a corpus of articles about various softwares and services as well as step-by-step instructions for common IT tasks.  They are discoverable in the [Princeton Service Portal](https://princeton.service-now.com/service) via the search function.

Instructions for how to create a KB article [can be found on Confluence here](https://pul-confluence.atlassian.net/wiki/spaces/IT/pages/1017282570/How+to+Create+a+Knowledge+Base+Article).
